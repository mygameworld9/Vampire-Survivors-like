# 模块: `components` (视图层)

该目录是应用程序的**视图**层，遵循数据-逻辑-视图 (DLV) 架构。其唯一职责是使用 React 渲染用户界面 (UI)。

此模块中的组件通常是“哑”组件，意味着它们通过 props 接收数据并进行渲染，自身不包含复杂的游戏逻辑。它们通过回调函数将用户操作传达回主 `GameComponent`。

---

## 组件详解

### `HUD.tsx`
-   **职责:** 渲染游戏主界面——抬头显示器 (HUD)。
-   **显示内容:** 玩家的生命值和经验条、当前等级、游戏计时器以及武器和技能的物品栏。

### `LevelUpModal.tsx`
-   **职责:** 渲染玩家升级时出现的模态窗口。
-   **功能:** 显示三个随机的升级选项，并将玩家的选择传达回游戏引擎。

### `PauseMenu.tsx`
-   **职责:** 渲染游戏暂停时出现的菜单。
-   **功能:** 提供恢复、查看武器/技能信息、重新开始游戏或返回主菜单的选项。

### `WeaponsPanel.tsx` & `SkillsPanel.tsx`
-   **职责:** 可从暂停菜单访问的信息面板。
-   **功能:**
    -   `WeaponsPanel`: 显示所有当前装备武器的详细统计数据。
    -   `SkillsPanel`: 显示所有已获得的主动和被动技能的详细信息。
