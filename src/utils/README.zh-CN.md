# 模块: `utils` (工具)

该目录包含在应用程序的多个模块中共享的辅助代码、类型定义和工具类。它本身不包含任何核心游戏逻辑，但提供了支持代码库其余部分的基本工具。

---

## 工具文件

### `types.ts`
-   **职责:** 为整个项目提供集中的TypeScript接口和类型定义。
-   **目的:**
    -   确保类型安全和一致性。例如，`IWeaponData` 接口保证了 `data/weaponData.ts` 中所有的武器配置对象都具有 `Weapon.ts` 实体类所期望的相同结构。
    -   定义了引擎中使用的复杂对象的结构，例如 `UpgradeOption`、`UpgradeLevel` 和 `IPlayerState`。

### `Vector2D.ts`
-   **职责:** 一个用于执行2D向量操作的简单数学类。
-   **目的:**
    -   提供一个清晰可复用的API来处理位置、方向和速度。
    -   包括基本的向量方法，如 `normalize()`，这对于计算移动方向至关重要。
    -   这个类被几乎所有的游戏实体广泛用于位置和移动计算。
