## ⚠️ 开发注意事项

1. **新武器类型需扩展 `Weapon.ts`**
   - ORBITING: 环绕逻辑
   - CHAIN: 弹跳目标查找
   - TRAP: 放置 + 触发机制

2. **新状态效果需扩展 `statusEffects.ts`**
   - POISON: DoT + 可叠加
   - STUN: 停止移动 + AI暂停

3. **新技能机制**
   - 暴击需修改伤害计算流程
   - 护盾需在 Player 添加护盾层属性
   - 动能需追踪移动状态
