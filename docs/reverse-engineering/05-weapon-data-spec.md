# ⚔️ 武器数值规格文档

> 本文档详细记录了游戏中所有 29 种武器的完整参数配置。

---

## 📌 概述

| 统计项 | 数值 |
|:---|:---:|
| 武器总数 | 29 |
| 武器类型 | 10 种 |
| 武器线 | 7 条 (含进化路径) |
| 最高进化等级 | Tier 3 |

## 🔫 武器类型说明

| 类型 | 英文标识 | 描述 | 代表武器 |
|:---|:---|:---|:---|
| 投射物 | `PROJECTILE` | 直线飞行的子弹 | 手枪、加特林 |
| 回旋镖 | `BOOMERANG` | 飞出后返回 | 回旋镖、音速飞碟 |
| 光环 | `AURA` | 持续范围伤害 | 烈阳、超新星 |
| 激光 | `LASER` | 瞬发直线范围 | 冰晶碎片 |
| 追踪 | `HOMING_PROJECTILE` | 自动锁定敌人 | 魔法飞弹 |
| 闪电 | `LIGHTNING` | 瞬发AOE | 雷杖 |
| 近战 | `MELEE` | 近身斩击 | 武士刀、毒匕首 |
| 环绕 | `ORBITING` | 环绕玩家旋转 | 灵魂宝珠 |
| 链式 | `CHAIN` | 弹跳攻击多目标 | 链电 |
| 陷阱 | `TRAP` | 放置地雷 | 尖刺陷阱 |

## 🏷️ 武器标签系统

每个武器都有标签用于协同效果：

| 标签 | 含义 | 典型武器 |
|:---|:---|:---|
| `PROJECTILE` | 投射物类 | 手枪、魔法飞弹 |
| `AREA` | 范围伤害 | 光环、闪电 |
| `MELEE` | 近战 | 武士刀、毒匕首 |
| `PHYSICAL` | 物理伤害 | 手枪、回旋镖 |
| `MAGIC` | 魔法伤害 | 魔法飞弹、灵魂宝珠 |
| `FIRE` | 火属性 | 烈阳、超新星 |
| `ICE` | 冰属性 | 冰晶碎片、寒霜地雷 |
| `LIGHTNING` | 雷属性 | 雷杖、链电 |
| `POISON` | 毒属性 | 毒匕首、瘟疫镰刀 |
| `DARK` | 暗属性 | 虚空吞噬者、虚空裂隙 |
