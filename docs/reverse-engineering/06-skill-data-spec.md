## 🧮 技能效果计算公式

### 动量系统

```
移动时:
  每帧积累 = 移动距离 × 系数
  最大层数 = 100
  伤害加成 = 层数 × 0.5%
  最大伤害加成 = 50%

静止时:
  每秒衰减 = 10层
```

### 暴击系统

```
暴击判定:
  随机数 < 暴击率 → 触发暴击
  
暴击伤害:
  最终伤害 = 基础伤害 × 暴击倍率
  默认: 基础伤害 × 1.8
```

### 护盾系统

```
护盾生成:
  间隔 = 10秒
  层数 = 1 (可升级)
  
护盾效果:
  完全吸收一次伤害，无论伤害值多少
  吸收后消耗1层护盾
```

### 吸血系统

```
击杀敌人时:
  回复生命 = 最大生命值 × 0.5%
  
示例:
  最大生命 = 100
  击杀1只怪 → 回复 0.5 HP
```
